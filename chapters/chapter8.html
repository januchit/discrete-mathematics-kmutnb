8.1

Relations on Sets

ความสัมพันธ์บนเซต, นิยาม, การแทนด้วย Matrix, Inverse

นิยามและสัญลักษณ์

นิยาม: Binary Relation (ความสัมพันธ์ไบนารี)

ให้ $A$ และ $B$ เป็นเซต ความสัมพันธ์จาก $A$ ไป $B$ คือเซตย่อยของผลคูณคาร์ทีเซียน:

$$R \subseteq A \times B$$
  • ถ้า $(a, b) \in R$ เขียนว่า $a \mathrel{R} b$ (อ่านว่า "$a$ สัมพันธ์กับ $b$")
  • ถ้า $(a, b) \notin R$ เขียนว่า $a \not\mathrel{R} b$

Relation on a set: กรณีพิเศษ $A = B$ ได้ $R \subseteq A \times A$ เรียกว่า "ความสัมพันธ์บน $A$" — เป็นกรณีที่นำไปสู่สมบัติสำคัญต่าง ๆ

ความสัมพันธ์ต่างจากฟังก์ชันตรงที่: สมาชิก $a \in A$ สามารถสัมพันธ์กับ $b \in B$ ได้ หลายตัว หรือ ไม่มีตัวใดเลย ก็ได้ ขณะที่ฟังก์ชันกำหนดค่าเดียวให้ทุก element ใน Domain

ตัวอย่าง 8.1.1 ความสัมพันธ์บนเซตจำกัด

ให้ $A = \{1, 2, 3\}$, $B = \{a, b, c\}$ และ $R = \{(1,a),(2,b),(2,c),(3,c)\}$

  • $(1,a) \in R$ → $1 \mathrel{R} a$ ✓
  • $(2,b),(2,c) \in R$ → 2 สัมพันธ์กับ สองตัว (ไม่ใช่ฟังก์ชัน)
  • $(3,c) \in R$ → $3 \mathrel{R} c$ ✓ แต่ 3 ไม่สัมพันธ์กับ $a$ หรือ $b$

Domain, Codomain, Range และ Inverse Relation

นิยาม: Domain, Codomain, Range

ให้ $R \subseteq A \times B$:

  • Domain: $\mathrm{dom}(R) = \{a \in A \mid \exists b \in B,\; (a,b) \in R\}$ — elements ของ $A$ ที่ปรากฏในความสัมพันธ์
  • Codomain: เซต $B$ ที่กำหนดไว้ทั้งหมด
  • Range: $\mathrm{range}(R) = \{b \in B \mid \exists a \in A,\; (a,b) \in R\}$ — elements ของ $B$ ที่ถูกสัมพันธ์จริง

อาจเกิดได้ว่า $\mathrm{dom}(R) \subsetneq A$ หรือ $\mathrm{range}(R) \subsetneq B$

นิยาม: Inverse Relation (ความสัมพันธ์ผกผัน)

ให้ $R \subseteq A \times B$ ความสัมพันธ์ผกผัน $R^{-1} \subseteq B \times A$ นิยามโดย:

$$R^{-1} = \{(b, a) \mid (a, b) \in R\}$$

กล่าวคือ $b \mathrel{R^{-1}} a$ ก็ต่อเมื่อ $a \mathrel{R} b$

สมบัติเมทริกซ์: $M_{R^{-1}} = M_R^T$ (Transpose ของเมทริกซ์ $R$)

สมบัติ: $(R^{-1})^{-1} = R$ และ $\mathrm{dom}(R^{-1}) = \mathrm{range}(R)$

ตัวอย่าง 8.1.2 Domain, Range และ Inverse

จาก $R = \{(1,a),(2,b),(2,c),(3,c)\}$ บน $\{1,2,3\} \times \{a,b,c\}$

Domain: $\{1, 2, 3\}$ (ทุก element ของ $A$ ปรากฏ)

Range: $\{a, b, c\}$ (ทุก element ของ $B$ ปรากฏ)

Inverse: $R^{-1} = \{(a,1),(b,2),(c,2),(c,3)\}$

$c \mathrel{R^{-1}} 2$ และ $c \mathrel{R^{-1}} 3$ — $c$ สัมพันธ์กับสองตัว แสดงว่า $R^{-1}$ ก็ไม่ใช่ฟังก์ชัน

การแทนความสัมพันธ์ด้วยเมทริกซ์

นิยาม: Relation Matrix (เมทริกซ์ 0/1)

ให้ $A = \{a_1,\ldots,a_m\}$ และ $B = \{b_1,\ldots,b_n\}$ เมทริกซ์แทนความสัมพันธ์ $M_R$ ขนาด $m \times n$:

$$M_R[i][j] = \begin{cases} 1 & \text{ถ้า } (a_i, b_j) \in R \\ 0 & \text{ถ้า } (a_i, b_j) \notin R \end{cases}$$

เรียกว่า boolean matrix หรือ zero-one matrix

  • จำนวน 1 ในเมทริกซ์ = $|R|$ (ขนาดของความสัมพันธ์)
  • แถวที่เป็น 0 ทั้งหมด → $a_i \notin \mathrm{dom}(R)$
  • คอลัมน์ที่เป็น 0 ทั้งหมด → $b_j \notin \mathrm{range}(R)$

8.2

Properties of Relations

Reflexive, Irreflexive, Symmetric, Antisymmetric, Transitive

ความสัมพันธ์บน $A$ ($R \subseteq A \times A$) อาจมีสมบัติพิเศษต่าง ๆ ที่ช่วยจำแนกประเภทและนำไปสู่ทฤษฎีต่าง ๆ เช่น Equivalence Relation และ Partial Order

5 สมบัติหลักของความสัมพันธ์

1. Reflexive (สะท้อนตัวเอง)

$$\forall x \in A,\; (x,x) \in R$$

ทุก element สัมพันธ์กับตัวเอง — diagonal ของเมทริกซ์เป็น 1 ทั้งหมด
ตัวอย่าง: $=$, $\leq$, $\mid$, $\subseteq$

2. Irreflexive (ไม่สะท้อน)

$$\forall x \in A,\; (x,x) \notin R$$

ไม่มี element สัมพันธ์กับตัวเอง — diagonal เป็น 0 ทั้งหมด
ตัวอย่าง: $<$, $>$, $\neq$, "เป็นพ่อของ"

3. Symmetric (สมมาตร)

$$\forall x,y \in A,\; (x,y)\in R \Rightarrow (y,x)\in R$$

เมทริกซ์สมมาตร: $M_R = M_R^T$
ตัวอย่าง: $=$, $\neq$, "เป็นพี่น้องกัน"

4. Antisymmetric (ปฏิสมมาตร)

$$(x,y)\in R \land (y,x)\in R \Rightarrow x=y$$

ถ้า $x$ สัมพันธ์ $y$ และ $y$ สัมพันธ์ $x$ ต้อง $x=y$
ตัวอย่าง: $\leq$, $\mid$, $\subseteq$

5. Transitive (การถ่ายทอด)

$$(x,y)\in R \land (y,z)\in R \Rightarrow (x,z)\in R$$

$x \to y \to z$ นัยถึง $x \to z$
ตัวอย่าง: $=$, $\leq$, $<$, $\mid$, $\subseteq$

ข้อสังเกตสำคัญ: Symmetric และ Antisymmetric ไม่ใช่คู่ตรงข้ามกัน ความสัมพันธ์อาจ:
  • เป็นทั้ง Symmetric และ Antisymmetric พร้อมกัน → เช่น $= \; $ (เฉพาะ identity pairs)
  • เป็น Symmetric แต่ไม่ Antisymmetric → เช่น "เป็นพี่น้องกัน"
  • เป็น Antisymmetric แต่ไม่ Symmetric → เช่น $\leq$
  • ไม่เป็นทั้งสองอย่าง → เช่น $R = \{(1,2),(2,1),(1,3)\}$

การพิสูจน์สมบัติ: $\leq$ บน $\mathbb{R}$

เราจะพิสูจน์ทั้ง 3 สมบัติของ $\leq$ และแสดงตัวอย่างโต้แย้งสำหรับ $\neq$

📋 การพิสูจน์: $\leq$ บน $\mathbb{R}$ เป็น Reflexive
กำหนดให้

ให้ $x \in \mathbb{R}$ เป็น element ใดก็ได้

อนุมาน

จากสมบัติของจำนวนจริง: $x = x$ ดังนั้น $x \leq x$

สรุป

$\forall x \in \mathbb{R},\; x \leq x$ ดังนั้น $\leq$ เป็น Reflexive ✓

∎
📋 การพิสูจน์: $\leq$ บน $\mathbb{R}$ เป็น Antisymmetric
กำหนดให้

ให้ $x, y \in \mathbb{R}$ สมมติว่า $x \leq y$ และ $y \leq x$

อนุมาน

จาก $x \leq y$: $y - x \geq 0$   ①

จาก $y \leq x$: $x - y \geq 0$ นั่นคือ $y - x \leq 0$   ②

จาก ① และ ②: $y - x = 0$ ดังนั้น $x = y$

สรุป

$(x \leq y) \land (y \leq x) \Rightarrow x = y$ ดังนั้น $\leq$ เป็น Antisymmetric ✓

∎
📋 การพิสูจน์: $\leq$ บน $\mathbb{R}$ เป็น Transitive
กำหนดให้

ให้ $x, y, z \in \mathbb{R}$ สมมติว่า $x \leq y$ และ $y \leq z$

อนุมาน

จาก $x \leq y$: $y - x \geq 0$

จาก $y \leq z$: $z - y \geq 0$

บวกกัน: $(y-x)+(z-y) = z-x \geq 0$ ดังนั้น $x \leq z$

สรุป

$(x \leq y) \land (y \leq z) \Rightarrow x \leq z$ ดังนั้น $\leq$ เป็น Transitive ✓

∎
ตัวอย่าง 8.2.1 ตัวอย่างโต้แย้ง: $\neq$ บน $\mathbb{Z}$ ไม่เป็น Transitive

แสดงว่าความสัมพันธ์ $\neq$ ไม่มีสมบัติ Transitive

ตัวอย่างโต้แย้ง: ให้ $x = 1$, $y = 2$, $z = 1$

  • $1 \neq 2$ ✓ → $(1,2) \in R_{\neq}$
  • $2 \neq 1$ ✓ → $(2,1) \in R_{\neq}$
  • แต่ $1 = 1$ → $1 \not\neq 1$ ✗ → $(1,1) \notin R_{\neq}$

เราพบ $(x,y),(y,z) \in R_{\neq}$ แต่ $(x,z) \notin R_{\neq}$ → $\neq$ ไม่เป็น Transitive

สังเกต: $\neq$ เป็น Symmetric (ถ้า $x \neq y$ แล้ว $y \neq x$) และ Irreflexive แต่ไม่เป็น Transitive

8.3

Equivalence Relations

ความสัมพันธ์สมมูล, Equivalence Classes, Partition Theorem

นิยาม Equivalence Relation

นิยาม: Equivalence Relation (ความสัมพันธ์สมมูล)

ความสัมพันธ์ $R$ บน $A$ เรียกว่า equivalence relation ก็ต่อเมื่อมีสมบัติครบ 3 ข้อ:

  1. Reflexive: $\forall x \in A,\; xRx$
  2. Symmetric: $\forall x,y \in A,\; xRy \Rightarrow yRx$
  3. Transitive: $\forall x,y,z \in A,\; xRy \land yRz \Rightarrow xRz$

มักเขียนด้วยสัญลักษณ์ $\sim$ หรือ $\equiv$ เพื่อสื่อถึงความ "เหมือนกันในแง่มุมใดแง่มุมหนึ่ง"

Equivalence Class (คลาสสมมูล)

นิยาม: Equivalence Class

ให้ $\sim$ เป็น equivalence relation บน $A$ และ $a \in A$

$$[a] = \{x \in A \mid x \sim a\}$$

คือเซตของสมาชิกทั้งหมดใน $A$ ที่สมมูลกับ $a$

ตัวแทน (representative): สมาชิกใดก็ได้ใน $[a]$ ใช้เป็นตัวแทน: $[a] = [b]$ เมื่อไรก็ตามที่ $a \sim b$

ทฤษฎีบท: สมบัติพื้นฐานของ Equivalence Classes

ให้ $\sim$ เป็น equivalence relation บน $A$ และ $a, b \in A$:

  1. $a \in [a]$ (เพราะ Reflexive)
  2. $a \sim b \Leftrightarrow [a] = [b]$
  3. $a \not\sim b \Leftrightarrow [a] \cap [b] = \emptyset$

พิสูจน์ข้อ 2 ($\Rightarrow$): สมมติ $a \sim b$ ถ้า $x \in [a]$ แล้ว $x \sim a$ โดย Transitive กับ $a \sim b$: $x \sim b$ → $x \in [b]$ ดังนั้น $[a] \subseteq [b]$ โดยอาร์กิวเมนต์สมมาตร (ใช้ $b \sim a$ จาก Symmetric) ได้ $[b] \subseteq [a]$ สรุป $[a] = [b]$ ✓

Partition Theorem

ทฤษฎีบท: Equivalence Relation ↔ Partition

ให้ $\sim$ เป็น equivalence relation บน $A$ เซตของ equivalence classes $A/{\sim} = \{[a] \mid a \in A\}$ เรียกว่า quotient set และเป็น partition ของ $A$:

  1. $[a] \neq \emptyset$ สำหรับทุก $a$ (เพราะ $a \in [a]$)
  2. $[a] \cap [b] = \emptyset$ หรือ $[a] = [b]$ สำหรับทุก $a,b$
  3. $\bigcup_{a \in A} [a] = A$

กลับกัน: partition ใดก็ตามของ $A$ นิยาม equivalence relation ได้: $x \sim y \Leftrightarrow x,y$ อยู่ใน part เดียวกัน

กล่าวคือ: equivalence relations บน $A$ สัมพันธ์แบบ bijection กับ partitions ของ $A$

ตัวอย่าง Equivalence Relations

ตัวอย่าง 8.3.1 Congruence mod 3 บน $\mathbb{Z}$

ตรวจสอบสมบัติ:

  • Reflexive: $3 \mid (a-a) = 0$ ✓
  • Symmetric: $3|(a-b) \Rightarrow 3|(b-a) = -(a-b)$ ✓
  • Transitive: $3|(a-b)$ และ $3|(b-c) \Rightarrow 3|(a-b+b-c) = (a-c)$ ✓

8.4

Modular Arithmetic

เลขคณิต Modulo, Congruence, Caesar Cipher, RSA

นิยาม Congruence Modulo n

นิยาม: Congruence Modulo $n$ (สมภาค)

ให้ $n \in \mathbb{Z}^+$ กล่าวว่า $a$ สมภาค $b$ modulo $n$ เขียนว่า:

$$a \equiv b \pmod{n}$$

ก็ต่อเมื่อ $n \mid (a - b)$

เทียบเท่า: $a \equiv b \pmod n$ ก็ต่อเมื่อ $a$ และ $b$ หาร $n$ แล้วได้เศษเหลือเท่ากัน

ตัวดำเนินการ mod: จาก Division Algorithm $a = qn + r$ ($0 \leq r < n$) ดังนั้น $a \bmod n = r$

ตัวอย่าง 8.4.1 ตัวอย่าง Congruence
  • $17 \equiv 5 \pmod 6$ เพราะ $6 \mid (17-5) = 12$ ✓
  • $23 \bmod 5 = 3$ เพราะ $23 = 4 \cdot 5 + 3$
  • $(-7) \bmod 3 = 2$ เพราะ $-7 = (-3) \cdot 3 + 2$ (เศษต้อง $\geq 0$)
  • $100 \equiv 4 \pmod 8$ เพราะ $100 = 12 \cdot 8 + 4$

การพิสูจน์: $\equiv \pmod n$ เป็น Equivalence Relation

📋 ทฤษฎีบท: $\equiv \pmod n$ เป็น Equivalence Relation บน $\mathbb{Z}$
Reflexive

ต้องการแสดง $a \equiv a \pmod n$: $a - a = 0 = 0 \cdot n$ ดังนั้น $n \mid 0$ ✓

Symmetric

สมมติ $a \equiv b \pmod n$, มี $k \in \mathbb{Z}$: $a - b = kn$

ดังนั้น $b - a = -kn = (-k)n$ และ $-k \in \mathbb{Z}$ → $n \mid (b-a)$ → $b \equiv a \pmod n$ ✓

Transitive

สมมติ $a \equiv b$ และ $b \equiv c \pmod n$, มี $k,l \in \mathbb{Z}$: $a-b=kn$, $b-c=ln$

$a - c = (a-b)+(b-c) = kn+ln = (k+l)n$ → $n \mid (a-c)$ → $a \equiv c \pmod n$ ✓

∎ ดังนั้น $\equiv \pmod n$ เป็น equivalence relation บน $\mathbb{Z}$

การคำนวณ mod n

ทฤษฎีบท: สมบัติการคำนวณ

ถ้า $a \equiv a' \pmod n$ และ $b \equiv b' \pmod n$ แล้ว:

  • $(a+b) \equiv (a'+b') \pmod n$
  • $(a-b) \equiv (a'-b') \pmod n$
  • $(a \cdot b) \equiv (a' \cdot b') \pmod n$

ผลสำคัญ: $((a \bmod n) + (b \bmod n)) \bmod n = (a+b) \bmod n$ — สามารถ reduce ก่อนหรือหลังบวกได้

ตัวอย่าง 8.4.2 Clock Arithmetic (mod 12)

นาฬิกาทำงานใน mod 12: ถ้าตอนนี้ 11 โมง อีก 4 ชั่วโมงจะเป็นกี่โมง?

$$11 + 4 = 15 \equiv 3 \pmod{12}$$

ดังนั้นจะเป็น 3 โมง

  • $10 + 5 \equiv 3 \pmod{12}$ (10 โมง + 5 ชั่วโมง = 3 โมง)
  • $3 - 7 \equiv -4 \equiv 8 \pmod{12}$ (ถอยหลัง 7 ชั่วโมงจาก 3 โมง = 8 โมงคืน)

Caesar Cipher — การเข้ารหัสด้วย mod 26

นิยาม: Caesar Cipher

เข้ารหัสตัวอักษร $m$ (A=0, B=1, …, Z=25) ด้วย shift key $k$:

$$\mathrm{enc}(m) = (m + k) \bmod 26$$ $$\mathrm{dec}(c) = (c - k) \bmod 26$$

การเข้ารหัสและถอดรหัสเป็น inverse กันโดยใช้สมบัติ modular arithmetic

Modular Exponentiation และ RSA

ตัวอย่าง 8.4.4 Modular Exponentiation: $3^5 \bmod 7$

วิธีตรง: $3^5 = 243 = 34 \times 7 + 5$ ดังนั้น $3^5 \equiv 5 \pmod 7$

Repeated Squaring (ลด mod ทุกขั้น — มีประสิทธิภาพกว่าสำหรับ exponent ขนาดใหญ่):

  • $3^1 \equiv 3 \pmod 7$
  • $3^2 \equiv 9 \equiv 2 \pmod 7$
  • $3^4 \equiv (3^2)^2 \equiv 2^2 = 4 \pmod 7$
  • $3^5 = 3^4 \cdot 3^1 \equiv 4 \cdot 3 = 12 \equiv 5 \pmod 7$ ✓
การประยุกต์ใช้ Modular Arithmetic ในวิทยาการคอมพิวเตอร์:
  • Hash Functions: $h(k) = k \bmod m$ — Hash Table ขนาด $m$ แปลง key เป็น index
  • Checksums: ISBN-10, barcode — ผลรวม mod ตรวจสอบความถูกต้องของข้อมูล
  • RSA Cryptography: เข้ารหัส $c \equiv m^e \pmod n$, ถอดรหัส $m \equiv c^d \pmod n$ — รากฐานของ HTTPS
  • Cyclic Buffers: Index ใน ring buffer = $(i+1) \bmod n$
  • Random Number Generation: Linear Congruential: $X_{n+1} = (aX_n+c) \bmod m$
8.5

Partial Order Relations

Partial Order, Hasse Diagram, Total Order, Extremal Elements

นิยาม Partial Order และ Poset

นิยาม: Partial Order (อันดับบางส่วน)

ความสัมพันธ์ $R$ บน $A$ เรียกว่า partial order ก็ต่อเมื่อมีสมบัติ:

  1. Reflexive: $\forall a \in A,\; a \mathrel{R} a$
  2. Antisymmetric: $a \mathrel{R} b \land b \mathrel{R} a \Rightarrow a = b$
  3. Transitive: $a \mathrel{R} b \land b \mathrel{R} c \Rightarrow a \mathrel{R} c$

คู่ $(A, R)$ เรียกว่า partially ordered set (poset) มักเขียน $a \preceq b$ แทน $aRb$

เขียน $a \prec b$ (strict) เมื่อ $a \preceq b$ และ $a \neq b$

Partial Order vs Equivalence:
  • Equivalence = Reflexive + Symmetric + Transitive → สร้าง partition
  • Partial Order = Reflexive + Antisymmetric + Transitive → สร้าง hierarchy

Hasse Diagram

นิยาม: Hasse Diagram

การวาด poset แบบกระชับโดย:

  1. ลบ self-loops: ไม่วาด $(a,a)$ เพราะ Reflexive นัยอยู่แล้ว
  2. ลบ transitive edges: ถ้า $a \prec b \prec c$ ไม่วาดเส้น $a \to c$ โดยตรง
  3. ทิศทาง = ความสูง: ถ้า $a \prec b$ วาง $b$ สูงกว่า $a$ — ไม่ต้องใส่หัวลูกศร

Cover relation: $a \prec_\mathrm{cov} b$ ก็ต่อเมื่อ $a \prec b$ และไม่มี $c$ ที่ $a \prec c \prec b$ — เส้นใน Hasse diagram แทน cover relation เท่านั้น

Total Order (Linear Order)

นิยาม: Total Order (อันดับเชิงเส้น)

Partial order $(A, \preceq)$ เป็น total order ก็ต่อเมื่อทุกคู่ comparable:

$$\forall a, b \in A,\; a \preceq b \;\text{ หรือ }\; b \preceq a$$

Hasse diagram ของ total order คือ เส้นตรง

ตัวอย่าง Total Order: $(\mathbb{Z}, \leq)$, $(\mathbb{R}, \leq)$, $(\{1,2,3\}, \leq)$

ตัวอย่าง Partial แต่ไม่ Total: $(\mathcal{P}(\{a,b\}), \subseteq)$ เพราะ $\{a\}$ และ $\{b\}$ ไม่ comparable

Extremal Elements

นิยาม: Maximal, Minimal, Greatest, Least

ใน poset $(A, \preceq)$:

  • Maximal element: $m \in A$ ที่ $\nexists a \in A,\; m \prec a$ — ไม่มี element ใหญ่กว่า
  • Minimal element: $m \in A$ ที่ $\nexists a \in A,\; a \prec m$ — ไม่มี element เล็กกว่า
  • Greatest element (maximum): $g \in A$ ที่ $\forall a \in A,\; a \preceq g$ — ใหญ่กว่าหรือเท่ากับทุกตัว (มีได้อย่างมาก 1 ตัว)
  • Least element (minimum): $l \in A$ ที่ $\forall a \in A,\; l \preceq a$ — เล็กกว่าหรือเท่ากับทุกตัว (มีได้อย่างมาก 1 ตัว)

ข้อแตกต่าง: Maximal element ไม่จำเป็นต้อง comparable กับทุกตัว — อาจมีหลาย maximal elements ได้; Greatest element ต้อง $\geq$ ทุกตัว — มีได้อย่างมาก 1 ตัว

ตัวอย่าง Partial Order

ตัวอย่าง 8.5.1 Divisibility บน $\{1,2,3,4,6,12\}$ — Hasse Diagram

ให้ $A = \{1,2,3,4,6,12\}$ กับ partial order $\mid$ (หารลงตัว)

Cover relations: $1 \prec_\mathrm{cov} 2$, $1 \prec_\mathrm{cov} 3$, $2 \prec_\mathrm{cov} 4$, $2 \prec_\mathrm{cov} 6$, $3 \prec_\mathrm{cov} 6$, $4 \prec_\mathrm{cov} 12$, $6 \prec_\mathrm{cov} 12$

  • Least element: 1 (หารทุกตัวลงตัว)
  • Greatest element: 12 (ถูกหารโดยทุกตัว)
  • Incomparable: 2 กับ 3, 4 กับ 3, 4 กับ 6 — ไม่ใช่ total order
ตัวอย่าง 8.5.2 Subset Relation บน $\mathcal{P}(\{a,b,c\})$

$(\mathcal{P}(\{a,b,c\}), \subseteq)$ เป็น partial order แต่ไม่ใช่ total order

Extremal elements:

  • Least element: $\emptyset$ (สับเซตของทุกเซต)
  • Greatest element: $\{a,b,c\}$ (ทุกเซตเป็นสับเซตของมัน)

Incomparable pairs: $\{a\}$ กับ $\{b\}$, $\{a,b\}$ กับ $\{a,c\}$, เป็นต้น

Hasse diagram มี 4 ระดับ: $\emptyset$ → singletons → pairs → $\{a,b,c\}$

ตัวอย่าง 8.5.3 Topological Sort — จัดลำดับงานตาม Dependencies

Topological sort หาลำดับ linear ที่สอดคล้องกับ partial order (เรียกว่า linear extension)

วิชาที่ต้องเรียนก่อน:

  • Calculus I $\prec$ Calculus II $\prec$ Real Analysis
  • Calculus I $\prec$ Differential Equations
  • Linear Algebra $\prec$ Real Analysis
  • Discrete Math $\prec$ Algorithms

Topological order ที่เป็นไปได้:

Discrete Math → Linear Algebra → Calculus I → Calculus II → Differential Equations → Real Analysis → Algorithms

Linear extension ไม่จำเป็นต้องมีเพียงแบบเดียว — อาจมีหลายแบบที่สอดคล้องกับ partial order

Algorithm: DFS-based topological sort ทำงานใน $O(V+E)$

แบบฝึกหัดแนะนำ

ฝึกทักษะ Relations, สมบัติ, Equivalence และ Partial Order

8.1

ให้ $A = \{1,2,3,4\}$ และ $R = \{(1,2),(1,4),(2,3),(3,4),(4,1)\}$ เป็นความสัมพันธ์บน $A$

  1. สร้างเมทริกซ์ 0/1 ของ $R$
  2. หา $R^{-1}$ ในรูปเซตของคู่อันดับ และเขียนเมทริกซ์ของ $R^{-1}$
  3. ตรวจสอบว่า $R$ เป็น Reflexive, Symmetric, Antisymmetric, Transitive หรือไม่ (พร้อมเหตุผล)
8.2

สำหรับความสัมพันธ์ต่อไปนี้บน $\mathbb{Z}$ ระบุสมบัติทั้ง 5 (Reflexive/Irreflexive/Symmetric/Antisymmetric/Transitive) พร้อมพิสูจน์หรือยกตัวอย่างโต้แย้ง:

  1. $R_1 = \{(a,b) \mid a + b \text{ เป็นเลขคู่}\}$
  2. $R_2 = \{(a,b) \mid |a - b| \leq 2\}$
  3. $R_3 = \{(a,b) \mid a^2 = b^2\}$
8.3

กำหนด $R$ บน $\mathbb{Z}$ โดย $a \mathrel{R} b$ ก็ต่อเมื่อ $a \equiv b \pmod 5$

  1. พิสูจน์ว่า $R$ เป็น equivalence relation
  2. หา equivalence classes ทั้งหมดของ $R$ บน $\mathbb{Z}$
  3. แสดงว่า equivalence classes เป็น partition ของ $\mathbb{Z}$
  4. ตัวแทนที่อยู่ใน $\{0,1,2,3,4\}$ ของแต่ละ class คืออะไร?
8.4

คำนวณและพิสูจน์โดยใช้สมบัติ modular arithmetic:

  1. $(2^{10} + 3^{10}) \bmod 5$ (힌트: คำนวณ $2^{10} \bmod 5$ และ $3^{10} \bmod 5$ แยกก่อน)
  2. $7^{100} \bmod 6$ (힌트: $7 \equiv 1 \pmod 6$)
  3. เข้ารหัส "MATH" ด้วย Caesar Cipher $k=13$ (ROT13) แล้วถอดรหัสกลับ
  4. พิสูจน์: ถ้า $a \equiv b \pmod n$ แล้ว $a^2 \equiv b^2 \pmod n$
8.5

พิจารณา $A = \{2,4,6,8,12,24\}$ กับความสัมพันธ์ $\mid$ (หารลงตัว)

  1. พิสูจน์ว่า $(A, \mid)$ เป็น partial order
  2. วาด Hasse Diagram ของ $(A, \mid)$
  3. หา Maximal, Minimal, Greatest, Least elements (ถ้ามี)
  4. ระบุคู่ที่ incomparable อย่างน้อย 3 คู่
  5. $(A, \mid)$ เป็น total order หรือไม่? อธิบาย
8.6

(ท้าทาย) กำหนด $S$ บน $\mathbb{Z}^+ \times \mathbb{Z}^+$ โดย $(a,b) \mathrel{S} (c,d)$ ก็ต่อเมื่อ $ad = bc$

  1. แสดงว่า $S$ เป็น equivalence relation (พิสูจน์ทั้ง 3 สมบัติ)
  2. อธิบายว่า equivalence class $[(a,b)]$ แทนอะไรในทางคณิตศาสตร์
  3. $(2,4)$, $(1,2)$, $(3,6)$ อยู่ใน equivalence class เดียวกันหรือไม่?
  4. Quotient set $(\mathbb{Z}^+ \times \mathbb{Z}^+)/S$ "ดูเหมือน" เซตอะไร?
