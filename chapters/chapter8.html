8.1

Relations on Sets

ความสัมพันธ์บนเซต, นิยาม, Matrix Representation

นิยามและสัญลักษณ์

นิยาม

ให้ $A$ และ $B$ เป็นเซต ความสัมพันธ์จาก $A$ ไป $B$ คือเซตย่อย $R \subseteq A \times B$ ถ้า $(a,b) \in R$ เขียนว่า $aRb$ (อ่านว่า "$a$ สัมพันธ์กับ $b$")

Relation on a set: กรณีพิเศษที่ $A = B$ จะได้ $R \subseteq A \times A$ เรียกว่า "ความสัมพันธ์บน $A$"

ตัวอย่าง 8.1.1

กำหนด $A = \{1,2,3\}$ และ $R = \{(1,2),(1,3),(2,3)\}$ บน $A$

จะเห็นว่า $1R2$, $1R3$, $2R3$ แต่ $(2,1) \notin R$ ดังนั้น $R$ ไม่ใช่ Symmetric

ความสัมพันธ์นี้สามารถมองได้ว่า "$a \, R \, b$ ก็ต่อเมื่อ $a < b$"

Matrix Representation

เมทริกซ์แทนความสัมพันธ์ (0/1 Matrix)

กำหนด $M_R$ เป็นเมทริกซ์ขนาด $|A| \times |B|$ โดย $(M_R)_{ij} = 1$ ถ้า $(a_i, b_j) \in R$ และ $= 0$ ถ้าไม่ใช่

สำหรับ $R = \{(1,2),(1,3),(2,3)\}$ บน $A = \{1,2,3\}$:

R123
1011
2001
3000

แถวคือ element ต้นทาง (domain), คอลัมน์คือ element ปลายทาง (codomain)

8.2

Properties of Relations

Reflexive, Symmetric, Antisymmetric, Transitive

4 สมบัติหลักของความสัมพันธ์

นิยาม

ให้ $R$ เป็นความสัมพันธ์บนเซต $A$ กำหนดสมบัติดังนี้:

Reflexive (การสะท้อน)

$\forall x \in A,\ (x,x) \in R$

ทุก element สัมพันธ์กับตัวเอง บนเมทริกซ์คือ diagonal ทั้งหมดเป็น 1

Symmetric (การสมมาตร)

$\forall x,y \in A,\ (x,y)\in R \Rightarrow (y,x)\in R$

เมทริกซ์สมมาตรตาม diagonal

Antisymmetric (การสมมาตรทางเดียว)

$(x,y)\in R \wedge (y,x)\in R \Rightarrow x = y$

คู่ $(x,y)$ และ $(y,x)$ อยู่ใน $R$ พร้อมกันได้ก็ต่อเมื่อ $x=y$

Transitive (การถ่ายทอด)

$(x,y)\in R \wedge (y,z)\in R \Rightarrow (x,z)\in R$

ถ้า $x$ ถึง $y$ และ $y$ ถึง $z$ ได้ ต้องถึง $z$ จาก $x$ ได้เสมอ

ตัวอย่าง 8.2.1 — พิสูจน์ $\leq$ บน $\mathbb{R}$

Reflexive: $\forall x \in \mathbb{R},\ x \leq x$  ✓

Antisymmetric: ถ้า $x \leq y$ และ $y \leq x$ ดังนั้น $x = y$  ✓

Transitive: ถ้า $x \leq y$ และ $y \leq z$ ดังนั้น $x \leq z$  ✓

Symmetric: $1 \leq 2$ แต่ $2 \not\leq 1$ ดังนั้น ไม่ใช่ Symmetric  ✗

ตารางสรุปสมบัติของความสัมพันธ์ทั่วไป

8.3

Equivalence Relations

ความสัมพันธ์สมมูล และ Equivalence Classes

นิยาม Equivalence Relation

นิยาม

ความสัมพันธ์ $R$ บนเซต $A$ เรียกว่า Equivalence Relation ก็ต่อเมื่อมีสมบัติครบทั้ง 3 ข้อ:

  1. Reflexive: $\forall x \in A,\ xRx$
  2. Symmetric: $xRy \Rightarrow yRx$
  3. Transitive: $xRy \wedge yRz \Rightarrow xRz$

Equivalence Classes และ Partition

Equivalence Class ของ $a$

$[a] = \{x \in A \mid xRa\}$ คือเซตของสมาชิกทั้งหมดที่สมมูลกับ $a$

ทฤษฎีบท: Equivalence classes ก่อให้เกิด partition ของ $A$ กล่าวคือ: ทุก class ไม่ว่าง, classes ไม่ทับซ้อนกัน, และ union ของ classes ทั้งหมดคือ $A$

ตัวอย่าง 8.3.1 — $\equiv \pmod{3}$ บน $\mathbb{Z}$

กำหนด $aRb$ ก็ต่อเมื่อ $3 \mid (a-b)$ ความสัมพันธ์นี้เป็น Equivalence Relation

ได้ equivalence classes 3 คลาส:

  • $[0] = \{\ldots,-6,-3,0,3,6,\ldots\}$ (หาร 3 ลงตัว)
  • $[1] = \{\ldots,-5,-2,1,4,7,\ldots\}$ (เหลือเศษ 1)
  • $[2] = \{\ldots,-4,-1,2,5,8,\ldots\}$ (เหลือเศษ 2)

$[0] \cup [1] \cup [2] = \mathbb{Z}$ และ classes ไม่ทับซ้อนกัน ยืนยันว่าเป็น partition

8.4

Modular Arithmetic

เลขคณิต Modulo และ Caesar Cipher

Congruence mod n

นิยาม

กำหนด $n \in \mathbb{Z}^+$ กล่าวว่า $a$ สมภาค $b$ modulo $n$ เขียน $a \equiv b \pmod{n}$ ก็ต่อเมื่อ $n \mid (a-b)$

การคำนวณ:

  • $(a + b) \bmod n = ((a \bmod n) + (b \bmod n)) \bmod n$
  • $(a \cdot b) \bmod n = ((a \bmod n) \cdot (b \bmod n)) \bmod n$
ตัวอย่าง 8.4.1

$17 \equiv 5 \pmod{6}$ เพราะ $6 \mid (17-5) = 12$  ✓

$(14 + 9) \bmod 5 = 23 \bmod 5 = 3$   หรือ   $(4 + 4) \bmod 5 = 8 \bmod 5 = 3$  ✓

Caesar Cipher — การเข้ารหัสด้วย Modular Arithmetic

สูตร: เข้ารหัสตัวอักษร $m$ (0–25) ด้วยกุญแจ $k$:   $c = (m + k) \bmod 26$
ถอดรหัส: $m = (c - k) \bmod 26$

8.5

Partial Order Relations

Partial Order, Hasse Diagram, Total Order

นิยาม Partial Order และ Poset

นิยาม

ความสัมพันธ์ $R$ บน $A$ เรียกว่า Partial Order ก็ต่อเมื่อ $R$ เป็น Reflexive, Antisymmetric และ Transitive

คู่ $(A, R)$ เรียกว่า Partially Ordered Set (Poset) เขียน $a \preceq b$ แทน $aRb$

Total Order: Partial order ที่ทุกคู่ $(a,b)$ เปรียบเทียบกันได้ ($a \preceq b$ หรือ $b \preceq a$)

ตัวอย่าง 8.5.1 — Divisibility บน $\{1,2,3,6\}$

กำหนด $a \mid b$ (หารลงตัว) เป็นความสัมพันธ์บน $A = \{1,2,3,6\}$

$R = \{(1,1),(2,2),(3,3),(6,6),(1,2),(1,3),(1,6),(2,6),(3,6)\}$

ตรวจสมบัติ: Reflexive ✓, Antisymmetric ✓, Transitive ✓ → เป็น Partial Order

แต่ 2 และ 3 เปรียบเทียบกันไม่ได้ ($2 \nmid 3$ และ $3 \nmid 2$) → ไม่ใช่ Total Order

Hasse Diagram

วิธีวาด Hasse Diagram

  • วาง element ที่ "ใหญ่กว่า" ไว้ด้านบน
  • ลากเส้นเฉพาะความสัมพันธ์ที่ "ติดกัน" (cover relation) — ไม่ต้องวาดลูกศรที่มาจาก transitivity หรือ reflexivity

Hasse diagram สำหรับ divisibility บน $\{1,2,3,4,6,12\}$:

1 อยู่ล่างสุด, 12 อยู่บนสุด เส้นแสดง cover relation

Relations — ความสัมพันธ์ครบวงจร

ประเด็นสำคัญที่ต้องจำ

  • $R \subseteq A \times B$ คือความสัมพันธ์ แทนด้วย 0/1 matrix หรือ graph
  • สมบัติ 4 ข้อ: Reflexive, Symmetric, Antisymmetric, Transitive
  • Equivalence Relation = Reflexive + Symmetric + Transitive → สร้าง partition
  • Partial Order = Reflexive + Antisymmetric + Transitive → วาด Hasse diagram
  • $a \equiv b \pmod{n}$ คือ equivalence relation ที่ใช้จริงใน cryptography
  • Total order คือ partial order ที่ทุกคู่เปรียบเทียบกันได้ (เช่น $\leq$ บน $\mathbb{R}$)

แบบฝึกหัดแนะนำ

ฝึกทำเพื่อความเข้าใจที่มั่นคง

  1. กำหนด $A = \{1,2,3,4\}$ และ $R = \{(a,b) \mid a \leq b\}$ เขียน $R$ เป็นเซต แล้วสร้าง 0/1 matrix แทน $R$
  2. พิสูจน์ว่า $R = \{(a,b) \in \mathbb{Z}^2 \mid 2 \mid (a-b)\}$ เป็น equivalence relation และระบุ equivalence classes ทั้งหมด
  3. ตรวจสอบว่า $R = \{(a,b) \mid a \text{ หาร } b \text{ ลงตัว}\}$ บน $\{2,4,8,16\}$ เป็น total order หรือ partial order
  4. คำนวณ $(7 \cdot 11) \bmod 13$ และ $(25 + 37) \bmod 8$ โดยใช้สมบัติ modular arithmetic
  5. ถอดรหัส "KHOOR" โดยใช้ Caesar cipher ด้วยกุญแจ $k = 3$
  6. วาด Hasse diagram สำหรับ divisibility บน $\{1,2,3,4,6,8,12,24\}$
