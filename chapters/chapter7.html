7.1

Functions Defined on General Sets

ฟังก์ชันบนเซตทั่วไป, นิยาม, สัญลักษณ์, Arrow Diagram

นิยามอย่างเป็นทางการ

นิยาม: ฟังก์ชัน (Function)

ให้ $A$ และ $B$ เป็นเซตที่ไม่ว่าง ฟังก์ชัน $f$ จาก $A$ ไป $B$ เขียนว่า $f: A \to B$ คือความสัมพันธ์ที่กำหนดค่า $f(a) \in B$ ให้กับสมาชิก $a \in A$ แต่ละตัว อย่างไม่กำกวม (exactly one value)

  • Domain (โดเมน): เซต $A$ ซึ่งเป็นเซตต้นทาง
  • Codomain (โคโดเมน): เซต $B$ ซึ่งเป็นเซตปลายทาง
  • Range / Image (รูป): $\mathrm{range}(f) = \{f(a) \mid a \in A\} \subseteq B$ — เซตของค่าที่ $f$ รับจริง ๆ (อาจเป็นสับเซตแท้ของ $B$)

สัญลักษณ์: $f(a)$ อ่านว่า "$f$ ของ $a$" หรือ "image ของ $a$ ภายใต้ $f$"

Arrow diagram ช่วยให้เห็นภาพได้ชัดเจน: ลูกศรจากแต่ละสมาชิกของ $A$ ชี้ไปยังสมาชิกของ $B$ ที่เป็นค่าของฟังก์ชัน

ตัวอย่าง: $f(1)=a,\ f(2)=c,\ f(3)=b,\ f(4)=c$ — Range $= \{a, b, c\}$

ตัวอย่าง 7.1.1 ฟังก์ชันจากเซตจำกัด

ให้ $A = \{1, 2, 3\}$, $B = \{x, y, z, w\}$ และ $f$ นิยามโดย $f(1) = y$, $f(2) = w$, $f(3) = y$

  • Domain $= A = \{1, 2, 3\}$
  • Codomain $= B = \{x, y, z, w\}$
  • Range $= \{y, w\}$ (เฉพาะค่าที่ถูก map ไปจริง)

สังเกต: $x$ และ $z$ อยู่ใน $B$ แต่ไม่อยู่ใน Range

Function Notation, Image และ Preimage

นิยาม: Image และ Preimage

ให้ $f: A \to B$ และ $a \in A$, $b \in B$

  • Image ของ $a$: ค่า $f(a) \in B$
  • Preimage ของ $b$: เซต $f^{-1}(\{b\}) = \{a \in A \mid f(a) = b\}$ — สมาชิกทั้งหมดใน $A$ ที่ map ไปยัง $b$

ถ้า $S \subseteq A$ แล้ว image ของ $S$ คือ $f(S) = \{f(a) \mid a \in S\}$

ถ้า $T \subseteq B$ แล้ว preimage ของ $T$ คือ $f^{-1}(T) = \{a \in A \mid f(a) \in T\}$

ตัวอย่าง 7.1.2 Image และ Preimage ของเซต

ให้ $f: \mathbb{Z} \to \mathbb{Z}$ นิยามโดย $f(n) = n^2$

  • $f(\{-2,-1,0,1,2\}) = \{0, 1, 4\}$ (image)
  • $f^{-1}(\{4\}) = \{-2, 2\}$ (preimage ของ 4)
  • $f^{-1}(\{3\}) = \emptyset$ (ไม่มีจำนวนเต็มที่ยกกำลัง 2 แล้วได้ 3)
  • Range $= \{0, 1, 4, 9, 16, \ldots\} = \{n^2 \mid n \in \mathbb{Z}\}$ ซึ่งเป็นสับเซตแท้ของ $\mathbb{Z}$

Well-Defined Functions และความเท่ากันของฟังก์ชัน

นิยาม: Well-Defined Function

ฟังก์ชัน $f: A \to B$ เป็น well-defined ก็ต่อเมื่อ: สำหรับทุก $a_1, a_2 \in A$ ถ้า $a_1 = a_2$ แล้ว $f(a_1) = f(a_2)$

กล่าวคือ สมาชิก $a$ แต่ละตัวต้องมีค่า $f(a)$ เพียงค่าเดียว (ไม่ใช่หลายค่า)

ตัวอย่าง 7.1.3 ฟังก์ชันที่ไม่ Well-Defined

นิยาม $g: \mathbb{Q} \to \mathbb{Z}$ โดย $g\!\left(\dfrac{p}{q}\right) = p$ (เอาตัวเศษ)

$g$ ไม่ well-defined เพราะ $\dfrac{1}{2} = \dfrac{2}{4}$ แต่ $g\!\left(\dfrac{1}{2}\right) = 1 \neq 2 = g\!\left(\dfrac{2}{4}\right)$

ค่าของ $g$ ขึ้นกับ การเขียน ของ $\dfrac{p}{q}$ ไม่ใช่ค่าของเศษส่วนเอง

นิยาม: ความเท่ากันของฟังก์ชัน (Equality of Functions)

ฟังก์ชัน $f: A \to B$ และ $g: A \to B$ เท่ากัน (เขียน $f = g$) ก็ต่อเมื่อ $f(a) = g(a)$ สำหรับทุก $a \in A$

สำคัญ: Domain และ Codomain ต้องเหมือนกัน ด้วย

ตัวอย่าง 7.1.4 ฟังก์ชันที่ดูเหมือนกันแต่ไม่เท่ากัน

พิจารณา $f: \mathbb{R} \to \mathbb{R}$ โดย $f(x) = |x|$ และ $g: \mathbb{R} \to \mathbb{R}$ โดย $g(x) = \sqrt{x^2}$

สำหรับทุก $x \in \mathbb{R}$: $f(x) = |x| = \sqrt{x^2} = g(x)$ ดังนั้น $f = g$ ✓

แต่ถ้า $h: [0,\infty) \to \mathbb{R}$ นิยามโดย $h(x) = x$ แล้ว $f \neq h$ เพราะ Domain ต่างกัน

ตัวอย่างฟังก์ชันสำคัญ

ฟังก์ชัน นิยาม Domain → Codomain ตัวอย่าง
Floor ⌊·⌋ $\lfloor x \rfloor =$ จำนวนเต็มที่มากที่สุด $\leq x$ $\mathbb{R} \to \mathbb{Z}$ $\lfloor 3.7 \rfloor = 3$, $\lfloor -1.2 \rfloor = -2$
Ceiling ⌈·⌉ $\lceil x \rceil =$ จำนวนเต็มที่น้อยที่สุด $\geq x$ $\mathbb{R} \to \mathbb{Z}$ $\lceil 3.2 \rceil = 4$, $\lceil -1.7 \rceil = -1$

ตัวอย่าง 7.1.5 Floor และ Ceiling บน Domain จำกัด

ให้ $A = \{x \in \mathbb{R} \mid 0 \leq x \leq 3\}$ และ $f: A \to \mathbb{Z}$ โดย $f(x) = \lfloor x \rfloor$

  • $f(0) = 0$, $f(0.5) = 0$, $f(1) = 1$, $f(2.9) = 2$, $f(3) = 3$
  • Range $= \{0, 1, 2, 3\}$
  • $f^{-1}(\{2\}) = [2, 3)$ — interval ของ $x$ ที่ $\lfloor x \rfloor = 2$
ตัวอย่าง 7.1.6 ฟังก์ชันบน Domain จำนวนเต็ม

ให้ $f: \mathbb{Z} \to \mathbb{Z}$ นิยามโดย $f(n) = 2n + 3$

  • $f(0) = 3$, $f(1) = 5$, $f(-1) = 1$, $f(5) = 13$
  • Range $= \{\ldots, -3, -1, 1, 3, 5, 7, \ldots\}$ = จำนวนคี่ทั้งหมด
  • $f^{-1}(\{7\}) = \{2\}$ เพราะ $2(2)+3 = 7$
หมายเหตุสำคัญ: Range $\subsetneq$ Codomain ได้ — เช่น $f(n) = 2n+3$ มี Codomain $\mathbb{Z}$ แต่ Range เป็นแค่จำนวนคี่ จำนวนคู่ทั้งหมดไม่ถูก map ถึง
7.2

One-to-One (Injective), Onto (Surjective), Bijective

สมบัติพื้นฐานของฟังก์ชันและฟังก์ชันผกผัน

Injective (One-to-One)

นิยาม: Injective Function (ฟังก์ชัน 1-1)

$f: A \to B$ เป็น injective (หรือ one-to-one) ก็ต่อเมื่อ

$$\forall a_1, a_2 \in A,\quad f(a_1) = f(a_2) \implies a_1 = a_2$$

หรือรูป contrapositive ที่มักใช้พิสูจน์:

$$a_1 \neq a_2 \implies f(a_1) \neq f(a_2)$$

ความหมาย: สมาชิกต่างกันใน $A$ จะถูก map ไปยังสมาชิกต่างกันใน $B$ — ไม่มี "การชน"

ตัวอย่าง 7.2.1 พิสูจน์ว่า $f(x) = 2x+1$ เป็น Injective บน $\mathbb{Z}$

ให้ $f: \mathbb{Z} \to \mathbb{Z}$ นิยามโดย $f(n) = 2n + 1$

🔍 การพิสูจน์
สมมติ

ให้ $n_1, n_2 \in \mathbb{Z}$ โดยที่ $f(n_1) = f(n_2)$

คำนวณ

$2n_1 + 1 = 2n_2 + 1$

$\Rightarrow 2n_1 = 2n_2$

$\Rightarrow n_1 = n_2$

สรุป

เนื่องจาก $f(n_1) = f(n_2) \Rightarrow n_1 = n_2$ ดังนั้น $f$ เป็น injective ✓

∎
ตัวอย่าง 7.2.2 พิสูจน์ว่า $g(n) = n^2$ ไม่ Injective บน $\mathbb{Z}$

Counter-example: $g(-2) = 4 = g(2)$ แต่ $-2 \neq 2$

ดังนั้น $g$ ไม่ injective เพราะสองสมาชิกต่างกัน ($-2$ และ $2$) map ไปยังค่าเดียวกัน

Surjective (Onto)

นิยาม: Surjective Function (Onto)

$f: A \to B$ เป็น surjective (หรือ onto) ก็ต่อเมื่อ

$$\forall b \in B,\ \exists a \in A \text{ such that } f(a) = b$$

ความหมาย: ทุกสมาชิกใน $B$ (Codomain) มี "preimage" อย่างน้อยหนึ่งตัวใน $A$ กล่าวคือ Range $=$ Codomain

ตัวอย่าง 7.2.3 พิสูจน์ว่า $f(n) = 2n+1$ บน $\mathbb{Z} \to \mathbb{Z}$ ไม่ Surjective

พิจารณา $b = 2 \in \mathbb{Z}$ (จำนวนคู่)

ต้องหา $n \in \mathbb{Z}$ ที่ $2n + 1 = 2$ กล่าวคือ $n = \dfrac{1}{2} \notin \mathbb{Z}$

ไม่มีจำนวนเต็ม $n$ ที่ $f(n) = 2$ ดังนั้น $f$ ไม่ surjective

Range ของ $f$ คือจำนวนคี่ทั้งหมด ซึ่งเป็นสับเซตแท้ของ $\mathbb{Z}$

ตัวอย่าง 7.2.4 Surjective โดยเปลี่ยน Codomain

ให้ $f: \mathbb{Z} \to \mathbb{Z}_{\text{odd}}$ นิยามโดย $f(n) = 2n + 1$ (Codomain = จำนวนคี่)

🔍 พิสูจน์ Surjective
ให้ $b$ เป็นจำนวนคี่ใดๆ

เขียน $b = 2k+1$ สำหรับ $k \in \mathbb{Z}$ (นิยามของจำนวนคี่)

หา Preimage

กำหนด $a = k \in \mathbb{Z}$ แล้ว $f(a) = 2k + 1 = b$

สรุป

สำหรับทุกจำนวนคี่ $b$ มี $a = k \in \mathbb{Z}$ ที่ $f(a) = b$ ดังนั้น $f$ เป็น surjective ✓

∎

Bijection และ Inverse Function

นิยาม: Bijection (การจับคู่แบบหนึ่งต่อหนึ่งและทั่วถึง)

$f: A \to B$ เป็น bijection (หรือ bijective function) ก็ต่อเมื่อ $f$ เป็นทั้ง injective และ surjective

  • แต่ละสมาชิกใน $B$ มี preimage ใน $A$ ตรงหนึ่งตัวพอดี
  • เรียกอีกชื่อว่า one-to-one correspondence

นิยาม: Inverse Function (ฟังก์ชันผกผัน)

ถ้า $f: A \to B$ เป็น bijection แล้ว inverse function $f^{-1}: B \to A$ นิยามโดย

$$f^{-1}(b) = a \iff f(a) = b$$

ทฤษฎีบท: $f^{-1}$ มีอยู่ก็ต่อเมื่อ $f$ เป็น bijection เท่านั้น

และ: $(f^{-1} \circ f)(a) = a$ สำหรับทุก $a \in A$, และ $(f \circ f^{-1})(b) = b$ สำหรับทุก $b \in B$

ตัวอย่าง 7.2.5 หา Inverse ของ $f(n) = 2n + 1$ บน $\mathbb{Z} \to \mathbb{Z}_{\text{odd}}$

เนื่องจาก $f$ เป็น bijection (injective จากตัวอย่าง 7.2.1, surjective จากตัวอย่าง 7.2.4)

หา $f^{-1}$: ให้ $b = 2n+1$ ดังนั้น $n = \dfrac{b-1}{2}$

$$f^{-1}(b) = \frac{b-1}{2}, \quad b \in \mathbb{Z}_{\text{odd}}$$

ตรวจสอบ: $f^{-1}(f(3)) = f^{-1}(7) = \dfrac{7-1}{2} = 3$ ✓

วิธีการพิสูจน์ Injective / Surjective / Bijective

สมบัติ วิธีพิสูจน์ว่าเป็นจริง วิธีแสดงว่าไม่เป็นจริง
Injective สมมติ $f(a_1) = f(a_2)$ แล้วพิสูจน์ว่า $a_1 = a_2$ หา counter-example: $a_1 \neq a_2$ แต่ $f(a_1) = f(a_2)$
Surjective ให้ $b \in B$ ใดๆ แล้วสร้าง $a \in A$ อย่างชัดเจนที่ $f(a) = b$ หา $b \in B$ ที่ไม่มี $a \in A$ ที่ $f(a) = b$
Bijective พิสูจน์ทั้ง injective และ surjective แสดงว่า injective หรือ surjective ล้มเหลว
ตัวอย่าง 7.2.6 ทดสอบ $f: \mathbb{R} \to \mathbb{R}$, $f(x) = x^3 - x$

Injective? $f(0) = 0$ และ $f(1) = 1 - 1 = 0$ ดังนั้น $f(0) = f(1) = 0$ แต่ $0 \neq 1$ → ไม่ injective

Surjective? $f$ เป็น polynomial ที่ $\lim_{x \to \pm\infty} f(x) = \pm\infty$ และ $f$ ต่อเนื่อง → โดย IVT, ทุก $b \in \mathbb{R}$ มี preimage → surjective

Bijective? ไม่ bijective เพราะไม่ injective

7.3

Composition of Functions

การประกอบฟังก์ชัน, Associativity, Identity

นิยาม Composition

นิยาม: Composition (การประกอบฟังก์ชัน)

ให้ $f: A \to B$ และ $g: B \to C$ (Codomain ของ $f$ เป็น Domain ของ $g$) Composition $g \circ f: A \to C$ นิยามโดย

$$(g \circ f)(x) = g(f(x)), \quad \forall x \in A$$

สังเกต: อ่าน $g \circ f$ ว่า "$g$ composed with $f$" หรือ "$g$ ของ $f$" — ทำ $f$ ก่อน แล้วจึงทำ $g$

ตัวอย่าง 7.3.1 คำนวณ Composition ของฟังก์ชัน

ให้ $f: \mathbb{R} \to \mathbb{R}$, $f(x) = 2x + 1$ และ $g: \mathbb{R} \to \mathbb{R}$, $g(x) = x^2$

$g \circ f$: $(g \circ f)(x) = g(f(x)) = g(2x+1) = (2x+1)^2 = 4x^2 + 4x + 1$

$f \circ g$: $(f \circ g)(x) = f(g(x)) = f(x^2) = 2x^2 + 1$

ข้อสังเกต: $g \circ f \neq f \circ g$ — Composition ไม่ใช่ commutative โดยทั่วไป!

ตรวจสอบ: $(g \circ f)(2) = (2 \cdot 2 + 1)^2 = 25$; $(f \circ g)(2) = 2(4) + 1 = 9$ ✓

ตัวอย่าง 7.3.2 Composition ของฟังก์ชันบนเซตจำกัด

ให้ $A = \{1,2,3\}$, $B = \{a,b,c\}$, $C = \{x,y\}$

$f: A \to B$: $f(1)=a$, $f(2)=b$, $f(3)=a$

$g: B \to C$: $g(a)=x$, $g(b)=y$, $g(c)=x$

$(g \circ f)(1) = g(f(1)) = g(a) = x$

$(g \circ f)(2) = g(f(2)) = g(b) = y$

$(g \circ f)(3) = g(f(3)) = g(a) = x$

สมบัติของ Composition

ทฤษฎีบท: Composition เป็น Associative

ให้ $f: A \to B$, $g: B \to C$, $h: C \to D$ แล้ว

$$h \circ (g \circ f) = (h \circ g) \circ f$$

หลักฐาน: สำหรับทุก $x \in A$: $(h \circ (g \circ f))(x) = h((g \circ f)(x)) = h(g(f(x)))$ และ $((h \circ g) \circ f)(x) = (h \circ g)(f(x)) = h(g(f(x)))$ ซึ่งเท่ากัน ∎

ทฤษฎีบท: Identity Function กับ Composition

ให้ $f: A \to B$ และ $\mathrm{id}_A: A \to A$, $\mathrm{id}_B: B \to B$ เป็น identity functions แล้ว

$$f \circ \mathrm{id}_A = f \qquad \text{และ} \qquad \mathrm{id}_B \circ f = f$$

ทฤษฎีบท: Inverse กับ Composition

ถ้า $f: A \to B$ เป็น bijection แล้ว

$$f^{-1} \circ f = \mathrm{id}_A \qquad \text{และ} \qquad f \circ f^{-1} = \mathrm{id}_B$$
ตัวอย่าง 7.3.3 ตรวจสอบ Inverse ด้วย Composition

ให้ $f: \mathbb{R} \to \mathbb{R}$, $f(x) = 3x - 2$ และ $f^{-1}(x) = \dfrac{x+2}{3}$

$(f^{-1} \circ f)(x) = f^{-1}(3x-2) = \dfrac{(3x-2)+2}{3} = \dfrac{3x}{3} = x$ ✓

$(f \circ f^{-1})(x) = f\!\left(\dfrac{x+2}{3}\right) = 3 \cdot \dfrac{x+2}{3} - 2 = x + 2 - 2 = x$ ✓

Composition รักษาสมบัติ Injective และ Surjective

ทฤษฎีบท: Composition ของ Injections เป็น Injection

ถ้า $f: A \to B$ และ $g: B \to C$ เป็น injective ทั้งคู่ แล้ว $g \circ f: A \to C$ เป็น injective

🔍 การพิสูจน์
สมมติ

ให้ $a_1, a_2 \in A$ โดยที่ $(g \circ f)(a_1) = (g \circ f)(a_2)$

กล่าวคือ $g(f(a_1)) = g(f(a_2))$

ใช้สมบัติ Injective ของ $g$

เนื่องจาก $g$ เป็น injective และ $g(f(a_1)) = g(f(a_2))$ ดังนั้น $f(a_1) = f(a_2)$

ใช้สมบัติ Injective ของ $f$

เนื่องจาก $f$ เป็น injective และ $f(a_1) = f(a_2)$ ดังนั้น $a_1 = a_2$

เราได้พิสูจน์แล้วว่า $(g \circ f)(a_1) = (g \circ f)(a_2) \Rightarrow a_1 = a_2$ ดังนั้น $g \circ f$ เป็น injective ✓

∎

ทฤษฎีบท: Composition ของ Surjections เป็น Surjection

ถ้า $f: A \to B$ และ $g: B \to C$ เป็น surjective ทั้งคู่ แล้ว $g \circ f: A \to C$ เป็น surjective

🔍 การพิสูจน์
ให้ $c \in C$ ใดๆ

ต้องการหา $a \in A$ ที่ $(g \circ f)(a) = c$

ใช้สมบัติ Surjective ของ $g$

เนื่องจาก $g: B \to C$ เป็น surjective, มี $b \in B$ ที่ $g(b) = c$

ใช้สมบัติ Surjective ของ $f$

เนื่องจาก $f: A \to B$ เป็น surjective, มี $a \in A$ ที่ $f(a) = b$

ดังนั้น $(g \circ f)(a) = g(f(a)) = g(b) = c$ ✓

สรุป: $g \circ f$ เป็น surjective ✓

∎
ผลลัพธ์ร่วม: ถ้า $f$ และ $g$ เป็น bijection ทั้งคู่ แล้ว $g \circ f$ เป็น bijection ด้วย และ $(g \circ f)^{-1} = f^{-1} \circ g^{-1}$
ตัวอย่าง 7.3.4 Composition ของ Bijections

ให้ $f(x) = x + 1$ และ $g(x) = 2x$ บน $\mathbb{R}$

ทั้ง $f$ และ $g$ เป็น bijection บน $\mathbb{R}$

$(g \circ f)(x) = g(x+1) = 2(x+1) = 2x + 2$

Inverse: $(g \circ f)^{-1}(x) = \dfrac{x}{2} - 1 = f^{-1}(g^{-1}(x))$ ✓

ตรวจสอบ: $f^{-1}(x) = x-1$, $g^{-1}(x) = \dfrac{x}{2}$

$(f^{-1} \circ g^{-1})(x) = f^{-1}\!\left(\dfrac{x}{2}\right) = \dfrac{x}{2} - 1$ ✓

7.4

Cardinality with Applications

ขนาดของเซต, Countable vs Uncountable, Cantor's Diagonal

Cardinality ของเซตจำกัด

นิยาม: Cardinality (Cardinality ของเซต)

Cardinality ของเซต $A$ (เขียน $|A|$) คือจำนวนสมาชิกของ $A$

  • $|\emptyset| = 0$
  • $|\{a, b, c\}| = 3$
  • $|\{1, 2, \ldots, n\}| = n$

เซตสองเซตมี Cardinality เท่ากัน ($|A| = |B|$) ก็ต่อเมื่อมี bijection $f: A \to B$

นิยามนี้ทำงานได้กับเซตอนันต์ด้วย — เป็นรากฐานของ Cardinality ของเซตอนันต์

ตัวอย่าง 7.4.1 Cardinality ของเซตจำกัด
  • $|\{2, 4, 6, 8, 10\}| = 5$
  • $|2^{\{1,2,3\}}| = 2^3 = 8$ (Power set ของ 3 สมาชิก)
  • สำหรับเซตจำกัด $A$ และ $B$: $|A \cup B| = |A| + |B| - |A \cap B|$ (Inclusion-Exclusion)

เซตอนันต์: Countable vs Uncountable

นิยาม: Countably Infinite

เซต $A$ เป็น countably infinite (อนันต์นับได้) ก็ต่อเมื่อมี bijection $f: \mathbb{N} \to A$ กล่าวคือ $|A| = |\mathbb{N}|$

เซตที่เป็น finite หรือ countably infinite เรียกรวมว่า countable

เซตที่ไม่ countable เรียกว่า uncountable

ทฤษฎีบท: $|\mathbb{N}| = |\mathbb{Z}|$

เซตจำนวนธรรมชาติ $\mathbb{N} = \{0, 1, 2, 3, \ldots\}$ และเซตจำนวนเต็ม $\mathbb{Z} = \{\ldots, -2, -1, 0, 1, 2, \ldots\}$ มี cardinality เท่ากัน

🔍 การพิสูจน์ด้วย Bijection แบบ Zigzag

สร้างการนับ $\mathbb{Z}$ โดยเรียงสลับ: $0, 1, -1, 2, -2, 3, -3, \ldots$

นิยาม $f: \mathbb{N} \to \mathbb{Z}$ โดย

$$f(n) = \begin{cases} \dfrac{n}{2} & \text{ถ้า } n \text{ เป็นเลขคู่} \\[6pt] -\dfrac{n+1}{2} & \text{ถ้า } n \text{ เป็นเลขคี่} \end{cases}$$
∎
$n \in \mathbb{N}$ 01234567
$f(n) \in \mathbb{Z}$ 0−11−22−33−4

ฟังก์ชัน $f$ เป็น bijection: ทุกจำนวนเต็มถูกนับถึงครบทุกตัว

ทฤษฎีบท: $|\mathbb{N}| = |\mathbb{Q}|$

เซตจำนวนตรรกยะ $\mathbb{Q}$ เป็น countably infinite แม้ว่าจะดู "หนาแน่น" กว่า $\mathbb{N}$ มาก

แนวคิดการพิสูจน์: จัดเรียงจำนวนตรรกยะบวกในรูป grid (แถวที่ $m$ คือตัวส่วน $m$, คอลัมน์ที่ $n$ คือตัวเศษ $n$) แล้วนับตาม diagonal zigzag — วิธีนี้เรียกว่า Cantor's enumeration of rationals

สรุป Countability: $|\mathbb{N}| = |\mathbb{Z}| = |\mathbb{Q}|$ — เซตทั้งสามนี้มี Cardinality เท่ากัน เรียก Cardinality นี้ว่า $\aleph_0$ (อ่านว่า "aleph-naught" หรือ "aleph-zero")

Cantor's Diagonal Argument: $|\mathbb{R}| > |\mathbb{N}|$

ทฤษฎีบท: $\mathbb{R}$ เป็น Uncountable

เซตจำนวนจริง $\mathbb{R}$ เป็น uncountable กล่าวคือ $|\mathbb{R}| > |\mathbb{N}|$ ไม่มี bijection จาก $\mathbb{N}$ ไป $\mathbb{R}$

🔍 Cantor's Diagonal Argument (พิสูจน์โดย Contradiction)
สมมติ Contradiction

สมมติว่า $\mathbb{R}$ เป็น countable กล่าวคือมี bijection $f: \mathbb{N} \to [0,1)$ (เพียงพอที่จะพิจารณา interval $[0,1)$)

ดังนั้นเราสามารถจัดเรียงจำนวนจริงใน $[0,1)$ เป็น list ได้:

$$r_0 = 0.d_{00}\,d_{01}\,d_{02}\,d_{03}\,\cdots$$ $$r_1 = 0.d_{10}\,d_{11}\,d_{12}\,d_{13}\,\cdots$$ $$r_2 = 0.d_{20}\,d_{21}\,d_{22}\,d_{23}\,\cdots$$ $$\vdots$$
สร้าง Diagonal Number

สร้างจำนวน $s = 0.s_0 s_1 s_2 s_3 \cdots$ โดย

$$s_n = \begin{cases} 5 & \text{ถ้า } d_{nn} \neq 5 \\ 6 & \text{ถ้า } d_{nn} = 5 \end{cases}$$

กล่าวคือ เลข $s_n$ ถูกเลือกให้แตกต่างจากเลขบน "Diagonal" $d_{nn}$

Contradiction

$s \in [0,1)$ เป็นจำนวนจริง แต่ $s \neq r_n$ สำหรับทุก $n$ เพราะ $s$ และ $r_n$ ต่างกันที่ตำแหน่งที่ $n$ (ทศนิยมตำแหน่งที่ $n$ คือ $s_n \neq d_{nn}$)

ดังนั้น $s$ ไม่อยู่ใน list ซึ่งขัดแย้งกับสมมติว่า $f$ เป็น bijection (surjection)

สรุป: $\mathbb{R}$ เป็น uncountable ✓

∎

ตัวอย่าง Diagonal Argument:

$r_0 = 0.$42178…
$r_1 = 0.3$7041…
$r_2 = 0.91$256…
$r_3 = 0.083$39…
$r_4 = 0.5602$1…

Diagonal digits (แดง): 4, 7, 2, 3, 1 → เลือก $s_n$ ต่างกัน → $s = 0.58546\ldots$

$s \neq r_n$ สำหรับทุก $n$ เพราะต่างกันที่ตำแหน่งที่ $n$

ความหมาย: Cardinality ของ $\mathbb{R}$ ใหญ่กว่า $|\mathbb{N}| = \aleph_0$ อย่างแท้จริง เรียก $|\mathbb{R}| = \mathfrak{c}$ ("continuum") ทฤษฎีนี้แสดงว่า "ระดับ" ของอนันต์มีหลายระดับ — สิ่งที่ Cantor ค้นพบเป็นครั้งแรกในปี ค.ศ. 1874

Cantor–Schröder–Bernstein Theorem

ทฤษฎีบท: Cantor–Schröder–Bernstein (CSB)

ถ้ามี injections $f: A \to B$ และ $g: B \to A$ แล้วมี bijection $h: A \to B$

เขียนในรูป Cardinality: ถ้า $|A| \leq |B|$ และ $|B| \leq |A|$ แล้ว $|A| = |B|$

(ทฤษฎีบทนี้ใช้พิสูจน์ที่ยากกว่า — แสดงเพียง statement เท่านั้น)

การประยุกต์ CSB

ทฤษฎีบท CSB ใช้เพื่อพิสูจน์ความเท่ากันของ Cardinality โดยไม่ต้องสร้าง bijection โดยตรง ซึ่งมักยาก แต่แค่ต้องหา injection ทั้งสองทิศทาง ซึ่งง่ายกว่า

ตัวอย่าง: $|(0,1)| = |\mathbb{R}|$

ใช้ CSB: $f: (0,1) \hookrightarrow \mathbb{R}$ คือ identity (injection ชัดเจน) และ $g: \mathbb{R} \hookrightarrow (0,1)$ คือ $g(x) = \dfrac{1}{\pi}\arctan(x) + \dfrac{1}{2}$ (injection เพราะ $\arctan$ เป็น strictly increasing) ดังนั้น $|(0,1)| = |\mathbb{R}|$

Hierarchy ของ Cardinality ของเซตทั่วไป

สำหรับเซต $A$ ใดๆ เสมอมี $|A| < |2^A|$ (Cantor's Theorem) ดังนั้นไม่มี "Cardinality สูงสุด":

$$|\mathbb{N}| < |\mathcal{P}(\mathbb{N})| < |\mathcal{P}(\mathcal{P}(\mathbb{N}))| < \cdots$$
📋

สรุปบทที่ 7

Functions: ฟังก์ชัน, สมบัติ, Composition, Cardinality

7.1 Functions

  • $f: A \to B$ — ทุก $a \in A$ มีค่า $f(a) \in B$ อย่างไม่กำกวม
  • Domain $A$, Codomain $B$, Range $\subseteq B$
  • Image ของ $a$: $f(a)$; Preimage ของ $b$: $f^{-1}(\{b\})$
  • Well-defined: $a_1 = a_2 \Rightarrow f(a_1) = f(a_2)$
  • Equality: $f = g \iff f(a) = g(a)$ สำหรับทุก $a \in A$
  • Floor, Ceiling, Absolute value, Identity, Constant

7.2 Inj / Sur / Bij

  • Injective: $f(a_1) = f(a_2) \Rightarrow a_1 = a_2$ (ไม่ชน)
  • Surjective: $\forall b \in B,\ \exists a \in A$ ที่ $f(a) = b$ (คลุมทั้ง $B$)
  • Bijective: inj + sur → matching 1-1 สมบูรณ์
  • $f^{-1}$ มีอยู่ $\iff$ $f$ เป็น bijection
  • $f \circ f^{-1} = \mathrm{id}_B$, $f^{-1} \circ f = \mathrm{id}_A$

7.3 Composition

  • $(g \circ f)(x) = g(f(x))$ — ทำ $f$ ก่อน
  • Associative: $h \circ (g \circ f) = (h \circ g) \circ f$
  • ไม่ Commutative: $g \circ f \neq f \circ g$ โดยทั่วไป
  • Inj $\circ$ Inj = Inj; Sur $\circ$ Sur = Sur
  • Bij $\circ$ Bij = Bij; $(g \circ f)^{-1} = f^{-1} \circ g^{-1}$

7.4 Cardinality

  • $|A| = |B| \iff$ มี bijection $A \to B$
  • $|\mathbb{N}| = |\mathbb{Z}| = |\mathbb{Q}| = \aleph_0$ (countable)
  • $|\mathbb{R}| = \mathfrak{c} > \aleph_0$ (uncountable)
  • Cantor Diagonal: $\mathbb{R}$ ไม่สามารถนับได้
  • CSB Theorem: inj สองทาง $\Rightarrow$ bijection
  • $|A| < |2^A|$ สำหรับเซต $A$ ใดๆ
✏️

7.1
Functions and Notation: ให้ $A = \{1, 2, 3, 4, 5\}$ และ $B = \{a, b, c, d\}$ และ $f: A \to B$ นิยามโดย $f(1)=a$, $f(2)=b$, $f(3)=b$, $f(4)=c$, $f(5)=d$ (1) หา Range ของ $f$ (2) หา $f^{-1}(\{b\})$ (3) ฟังก์ชัน $g: A \to B$ นิยามโดยตารางเป็น well-defined หรือไม่ ถ้า $g(1)=a$, $g(1)=b$ (คนละค่า)? (4) ให้ $h(x) = \lfloor 2x \rfloor$ บน $[0, 2]$ — หา Range ของ $h$
7.2
Injective และ Surjective: พิจารณาฟังก์ชันต่อไปนี้บน $\mathbb{Z} \to \mathbb{Z}$ แต่ละข้อ พิสูจน์หรือหา counter-example: (1) $f(n) = 3n - 5$ — injective? surjective? (2) $g(n) = n^2 - 4$ — injective? surjective? (3) $h(n) = \lfloor n/2 \rfloor$ — injective? surjective? (4) ระบุว่าฟังก์ชันใดเป็น bijection และหา inverse ถ้ามี
7.3
Inverse Function: ให้ $f: \mathbb{R} \to \mathbb{R}$ โดย $f(x) = \dfrac{3x - 1}{2}$ (1) พิสูจน์ว่า $f$ เป็น bijection (2) หา $f^{-1}(x)$ (3) ตรวจสอบว่า $(f \circ f^{-1})(x) = x$ และ $(f^{-1} \circ f)(x) = x$ (4) หา $(f \circ f)(x)$ และ $(f^{-1} \circ f^{-1})(x)$
7.4
Composition: ให้ $f(x) = x^2 + 1$, $g(x) = 2x - 3$, $h(x) = \sqrt{x}$ (ที่ Domain $[0,\infty)$) (1) คำนวณ $(g \circ f)(x)$ และ $(f \circ g)(x)$ (2) คำนวณ $(h \circ f)(x)$ — ระบุ Domain ที่เหมาะสม (3) พิสูจน์ว่า $g \circ f$ เป็น injective โดยใช้ทฤษฎีบทที่เรียน (ระบุสมบัติของ $f$ และ $g$ ก่อน) (4) หา $(g \circ f)^{-1}(x)$ ถ้า $f$ และ $g$ เป็น bijection
7.5
Countability: (1) แสดงว่าเซตจำนวนคู่ $2\mathbb{Z} = \{\ldots, -4, -2, 0, 2, 4, \ldots\}$ เป็น countably infinite โดยสร้าง bijection $f: \mathbb{N} \to 2\mathbb{Z}$ อย่างชัดเจน (2) แสดงว่าเซต $\{1/n \mid n \in \mathbb{N}, n \geq 1\}$ เป็น countable (3) ถ้า $A$ และ $B$ เป็น countable แล้ว $A \cup B$ เป็น countable หรือไม่? อธิบาย
7.6
Cantor's Diagonal Argument: (1) อธิบาย Cantor's Diagonal Argument ในแบบของตัวเอง — ทำไมจึงพิสูจน์ว่า $\mathbb{R}$ เป็น uncountable? (2) แสดงว่า interval $(0, 1)$ เป็น uncountable (hint: ใช้ argument เดียวกัน) (3) ถ้า $A$ เป็น uncountable และ $A \subseteq B$ แล้ว $B$ เป็น uncountable หรือไม่? พิสูจน์หรือ disprove (4) ใช้ Cantor's Theorem อธิบายว่าทำไม $|\mathcal{P}(\mathbb{N})| > |\mathbb{N}|$
