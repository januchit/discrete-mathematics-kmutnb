1.1

Variables

การใช้ตัวแปรในคณิตศาสตร์

ตัวแปร (Variable) คือสัญลักษณ์ที่แทนจำนวนหรือวัตถุที่ยังไม่ระบุค่า การใช้ตัวแปรมี 3 บทบาทหลัก

บทบาทของตัวแปร

  1. Placeholder (ตัวแทนค่าที่ไม่รู้) — เช่น "Is there a number $x$ such that $x^2 + 1 = 0$?"
  2. Generality (ความทั่วไป) — เช่น "For all real numbers $n$, if $n > 2$ then $n^2 > 4$"
  3. Reference (การอ้างอิง) — ให้ชื่อชั่วคราวเพื่ออ้างอิงซ้ำในการอ้างเหตุผล
ตัวอย่าง 1.1.1 Writing Sentences Using Variables

จงเขียนประโยคต่อไปนี้ให้อยู่ในรูปแบบทางการโดยใช้ตัวแปร

a. Are there numbers with the property that the sum of their squares equals the square of their sum?

b. Given any real number, its square is nonnegative.

วิธีทำ

a. Are there numbers $a$ and $b$ with the property that $a^2 + b^2 = (a+b)^2$?

b. For all real numbers $r$, $r^2 \geq 0$.

ประเภทของประพจน์ทางคณิตศาสตร์

ประพจน์ทางคณิตศาสตร์ที่สำคัญมี 3 ประเภท

Universal Statement
กล่าวว่าสมบัติหนึ่งเป็นจริงสำหรับสมาชิกทุกตัวในเซต
ตัวอย่าง: "All positive numbers are greater than zero."
Conditional Statement
กล่าวว่าถ้าสิ่งหนึ่งเป็นจริงแล้วสิ่งอื่นก็ต้องเป็นจริงด้วย
ตัวอย่าง: "If 378 is divisible by 18, then 378 is divisible by 6."
Existential Statement
กล่าวว่ามีอย่างน้อยหนึ่งสิ่งที่สมบัติเป็นจริง
ตัวอย่าง: "There is a prime number that is even."

Universal Conditional Statements

ประพจน์ที่เป็นทั้ง Universal และ Conditional พร้อมกัน สามารถเขียนได้หลายรูปแบบที่สมมูลกัน

ตัวอย่าง การเขียน Universal Conditional ในรูปแบบต่างๆ

พิจารณาประพจน์: "For all animals $a$, if $a$ is a dog, then $a$ is a mammal."

รูปแบบประพจน์
ดั้งเดิมFor all animals $a$, if $a$ is a dog, then $a$ is a mammal.
เน้น ConditionalIf an animal is a dog, then the animal is a mammal.
เน้น UniversalFor all dogs $a$, $a$ is a mammal.
ไม่มีตัวแปรAll dogs are mammals.
ตัวอย่าง 1.1.2 Rewriting a Universal Conditional Statement

เติมช่องว่างเพื่อเขียนประพจน์ "For all real numbers $x$, if $x$ is nonzero then $x^2$ is positive" ในรูปแบบต่างๆ

a. If a real number is nonzero, then its square is positive.

b. For all nonzero real numbers $x$, $x^2$ is positive.

c. If $x$ is a nonzero real number, then $x^2$ is positive.

d. The square of any nonzero real number is positive.

e. All nonzero real numbers have positive squares.

Universal Existential Statements

ประพจน์ที่ส่วนแรกเป็น Universal (สำหรับทุกตัว) และส่วนที่สองเป็น Existential (มีบางสิ่งอยู่)

ตัวอย่าง: "Every real number has an additive inverse."

สามารถเขียนใหม่เป็น: For all real numbers $r$, there is a real number $s$ such that $s + r = 0$.

Existential Universal Statements

ประพจน์ที่ส่วนแรกเป็น Existential และส่วนที่สองเป็น Universal

ตัวอย่าง: "There is a positive integer that is less than or equal to every positive integer."

สามารถเขียนใหม่เป็น: There is a positive integer $m$ with the property that for all positive integers $n$, $m \leq n$.

1.2

The Language of Sets

ภาษาของเซต

คำว่า เซต (Set) ถูกใช้เป็นศัพท์ทางคณิตศาสตร์อย่างเป็นทางการครั้งแรกในปี 1879 โดย Georg Cantor (1845–1918) ซึ่งนิยามเซตว่าเป็นเพียงกลุ่มของสมาชิก

สัญลักษณ์เซต (Set Notation)

สัญลักษณ์พื้นฐาน

  • $x \in S$ — $x$ เป็นสมาชิกของ $S$
  • $x \notin S$ — $x$ ไม่ได้เป็นสมาชิกของ $S$
  • $\{1, 2, 3\}$ — เซตที่มีสมาชิก 1, 2, 3 (Set-Roster Notation)
  • $\{1, 2, 3, \ldots, 100\}$ — เซตของจำนวนเต็ม 1 ถึง 100
  • $\{1, 2, 3, \ldots\}$ — เซตของจำนวนเต็มบวกทั้งหมด
Axiom of Extension: เซตถูกกำหนดโดยสมาชิกของมันเท่านั้น ลำดับและการซ้ำของสมาชิกไม่มีผล
ดังนั้น $\{1, 2, 3\} = \{3, 1, 2\} = \{1, 1, 2, 3, 3\}$
ตัวอย่าง 1.2.1 Using the Set-Roster Notation

ให้ $A = \{1, 2, 3\}$, $B = \{3, 1, 2\}$, $C = \{1, 1, 2, 3, 3, 3\}$

$A$, $B$, $C$ มีสมาชิกเหมือนกันทุกประการ คือ 1, 2, 3 ดังนั้น $A = B = C$

สังเกตว่า $\{0\} \neq 0$ เพราะ $\{0\}$ เป็นเซตที่มีสมาชิกหนึ่งตัว ส่วน $0$ คือจำนวน

เซต $\{1, \{1\}\}$ มีสมาชิก 2 ตัว คือ $1$ และ $\{1\}$

เซตสำคัญทางคณิตศาสตร์

สัญลักษณ์เซตตัวอย่างสมาชิก
$\mathbb{R}$ จำนวนจริงทั้งหมด (Real numbers) $-2.5,\ 0,\ \sqrt{2},\ \pi$
$\mathbb{Z}$ จำนวนเต็มทั้งหมด (Integers) (จาก Zahlen ในภาษาเยอรมัน) $\ldots, -2, -1, 0, 1, 2, \ldots$
$\mathbb{Q}$ จำนวนตรรกยะทั้งหมด (Rational numbers) $\frac{1}{2},\ -\frac{3}{4},\ 5$
$\mathbb{R}^+$ จำนวนจริงบวก $0.1,\ 1,\ 3.14$
$\mathbb{Z}^+$ จำนวนเต็มบวก $1, 2, 3, \ldots$
$\mathbb{Z}_{\text{nonneg}}$ จำนวนเต็มไม่ติดลบ $0, 1, 2, 3, \ldots$
ชื่อ Discrete Mathematics มาจากไหน?
เส้นจำนวนจริง (Real number line) เป็น continuous เพราะไม่มีช่องว่าง แต่จำนวนเต็ม $\mathbb{Z}$ เป็น discrete เพราะแต่ละตัวอยู่ห่างกัน — นี่คือที่มาของคำว่า "Discrete Mathematics"

Set-Builder Notation

นิยาม — Set-Builder Notation
$$\{x \in S \mid P(x)\}$$

อ่านว่า "the set of all $x$ in $S$ such that $P(x)$" หมายถึงเซตของสมาชิกทุกตัว $x$ ใน $S$ ที่ทำให้ $P(x)$ เป็นจริง

ตัวอย่าง 1.2.2 Using the Set-Builder Notation
นิพจน์ความหมาย
$\{x \in \mathbb{R} \mid -2 < x < 5\}$ ช่วงเปิด $(-2,\ 5)$ บนเส้นจำนวนจริง
$\{x \in \mathbb{Z} \mid -2 < x < 5\}$ $\{-1, 0, 1, 2, 3, 4\}$
$\{x \in \mathbb{Z}^+ \mid -2 < x < 5\}$ $\{1, 2, 3, 4\}$

Subsets (เซตย่อย)

นิยาม — Subset

ถ้า $A$ และ $B$ เป็นเซต เรากล่าวว่า $A$ เป็นเซตย่อยของ $B$ เขียนว่า $A \subseteq B$

$$A \subseteq B \quad \Longleftrightarrow \quad \forall x,\ (x \in A \Rightarrow x \in B)$$

$A$ เป็น proper subset ของ $B$ (เขียน $A \subsetneq B$) ถ้า $A \subseteq B$ และมีสมาชิกบางตัวใน $B$ ที่ไม่อยู่ใน $A$

ตัวอย่าง 1.2.3 Subsets

ให้ $A = \mathbb{Z}^+$, $B = \{n \in \mathbb{Z} \mid 0 \leq n \leq 100\}$, $C = \{100, 200, 300, 400, 500\}$

  • $B \subseteq A$ — เท็จ เพราะ $0 \in B$ แต่ $0 \notin A$ (0 ไม่ใช่จำนวนเต็มบวก)
  • $C \subsetneq A$ — จริง เพราะทุกตัวใน $C$ เป็นจำนวนเต็มบวก
  • $C \subseteq B$ — เท็จ เพราะ $200 \in C$ แต่ $200 \notin B$
  • $C \subseteq C$ — จริง เสมอ (ทุกเซตเป็นเซตย่อยของตัวเอง)
ข้อควรระวัง: ความแตกต่างระหว่าง $\in$ และ $\subseteq$
  • $2 \in \{1, 2, 3\}$ — จริง (2 เป็นสมาชิก)
  • $\{2\} \subseteq \{1, 2, 3\}$ — จริง ({2} เป็นเซตย่อย)
  • $\{2\} \in \{1, 2, 3\}$ — เท็จ (เซต {2} ไม่ใช่สมาชิก)
  • $2 \subseteq \{1, 2, 3\}$ — เท็จ (จำนวน 2 ไม่ใช่เซต)

Cartesian Products (ผลคูณคาร์ทีเชียน)

นิยาม — Ordered Pair & Cartesian Product

Ordered pair $(a, b)$ คือคู่อันดับที่ $a$ คือสมาชิกตัวแรก และ $b$ คือสมาชิกตัวที่สอง โดย $(a,b) = (c,d)$ ก็ต่อเมื่อ $a = c$ และ $b = d$

Cartesian product ของเซต $A$ และ $B$:

$$A \times B = \{(a, b) \mid a \in A \text{ and } b \in B\}$$
ตัวอย่าง 1.2.6 Cartesian Products

ให้ $A = \{1, 2, 3\}$ และ $B = \{u, v\}$

$A \times B = \{(1,u),(2,u),(3,u),(1,v),(2,v),(3,v)\}$ — มี $3 \times 2 = 6$ สมาชิก

$B \times A = \{(u,1),(u,2),(u,3),(v,1),(v,2),(v,3)\}$ — มี $2 \times 3 = 6$ สมาชิก

$B \times B = \{(u,u),(u,v),(v,u),(v,v)\}$ — มี $2 \times 2 = 4$ สมาชิก

$\mathbb{R} \times \mathbb{R}$ คือเซตของจุดทุกจุดในระนาบคาร์ทีเชียน (Cartesian plane)

1.3

The Language of Relations and Functions

ภาษาของความสัมพันธ์และฟังก์ชัน

Relations (ความสัมพันธ์)

นิยาม — Relation

ให้ $A$ และ $B$ เป็นเซต ความสัมพันธ์ $R$ จาก $A$ ไป $B$ คือ เซตย่อยของ $A \times B$

สำหรับ $(x, y) \in A \times B$: เราเขียน $x\ R\ y$ ถ้า $(x, y) \in R$ และเขียน $x\not{R}\ y$ ถ้า $(x, y) \notin R$

  • Domain ของ $R$ คือเซต $A$
  • Co-domain ของ $R$ คือเซต $B$
ตัวอย่าง 1.3.1 A Relation as a Subset

ให้ $A = \{1,2\}$, $B = \{1,2,3\}$ และนิยามความสัมพันธ์ $R$ จาก $A$ ไป $B$ ดังนี้:

$$(x, y) \in R \quad \Longleftrightarrow \quad \frac{x-y}{2} \in \mathbb{Z}$$

ตรวจแต่ละคู่ใน $A \times B$:

คู่อันดับ$(x-y)/2$เป็นจำนวนเต็ม?อยู่ใน $R$?
$(1,1)$$0/2 = 0$ใช่✓
$(1,2)$$-1/2$ไม่✗
$(1,3)$$-2/2 = -1$ใช่✓
$(2,1)$$1/2$ไม่✗
$(2,2)$$0/2 = 0$ใช่✓
$(2,3)$$-1/2$ไม่✗

ดังนั้น $R = \{(1,1),(1,3),(2,2)\}$

Functions (ฟังก์ชัน)

นิยาม — Function

ฟังก์ชัน $F$ จาก $A$ ไป $B$ คือความสัมพันธ์ที่มี domain $A$ และ co-domain $B$ และต้องเป็นไปตามเงื่อนไข:

  1. (ทุกตัวมีคู่) สำหรับทุก $x \in A$ มี $y \in B$ บางตัวที่ $(x,y) \in F$
  2. (คู่ไม่ซ้ำ) สำหรับทุก $x \in A$ ถ้า $(x,y) \in F$ และ $(x,z) \in F$ แล้ว $y = z$

ถ้า $F$ คือฟังก์ชัน ค่าเดียวที่ $x$ สัมพันธ์กัน เรียกว่า $F(x)$ (อ่านว่า "F ของ x")

ตัวอย่าง 1.3.4 Functions and Relations on Finite Sets

ให้ $A = \{2,4,6\}$, $B = \{1,3,5\}$ พิจารณาว่าสิ่งต่อไปนี้เป็นฟังก์ชันหรือไม่

a. $R = \{(2,5),(4,1),(4,3),(6,5)\}$

❌ ไม่ใช่ฟังก์ชัน — $(4,1)$ และ $(4,3)$ มีสมาชิกตัวแรกเหมือนกันแต่ตัวที่สองต่างกัน (ละเมิดเงื่อนไขข้อ 2)

b. $S$: $(x,y) \in S$ หมายความว่า $y = x + 1$ (สำหรับ $x \in A$, $y \in B$)

❌ ไม่ใช่ฟังก์ชัน — $6 \in A$ แต่ $6+1=7 \notin B$ (ละเมิดเงื่อนไขข้อ 1)

c. $T$: $T(2)=5$, $T(4)=1$, $T(6)=1$

✓ เป็นฟังก์ชัน — ทุกตัวใน $A$ มีคู่ และไม่มีสมาชิกใดมีค่ามากกว่าหนึ่ง (สังเกตว่า $T(4) = T(6) = 1$ ได้)
Vertical Line Test: กราฟใดเป็นฟังก์ชันก็ต่อเมื่อเส้นแนวตั้งทุกเส้นตัดกราฟนั้นที่จุดมากที่สุด 1 จุด
— วงกลม $x^2 + y^2 = 1$ ไม่ใช่ฟังก์ชัน (เส้นแนวตั้งบางเส้นตัด 2 จุด)
— เส้นตรง $y = x - 1$ เป็นฟังก์ชัน

Function Machines และตัวอย่างฟังก์ชัน

ฟังก์ชันสามารถมองเป็น "เครื่องจักร" ที่รับ input $x$ แล้วให้ output $f(x)$

ตัวอย่าง 1.3.6 Functions Defined by Formulas
Squaring function
$f: \mathbb{R} \to \mathbb{R}$
$f(x) = x^2$
$f(3) = 9$, $f(-2) = 4$
Successor function
$g: \mathbb{Z} \to \mathbb{Z}$
$g(n) = n + 1$
$g(5) = 6$, $g(-1) = 0$
Constant function
$h: \mathbb{Q} \to \mathbb{Z}$
$h(r) = 2$
$h(\frac{1}{3}) = 2$, $h(100) = 2$
ตัวอย่าง 1.3.7 Equality of Functions

ให้ $f(x) = |x|$ และ $g(x) = \sqrt{x^2}$ สำหรับทุก $x \in \mathbb{R}$

เนื่องจาก $|x| = \sqrt{x^2}$ สำหรับทุกจำนวนจริง $x$ ดังนั้น $f = g$

Speaking Mathematically

1.1 Variables

  • ตัวแปรใช้แทนค่าที่ไม่รู้ เพื่อความทั่วไป หรือเพื่ออ้างอิง
  • Universal statement: "for all..."
  • Conditional statement: "if... then..."
  • Existential statement: "there exists..."
  • ประพจน์ผสม: Universal Conditional, Universal Existential, Existential Universal

1.2 The Language of Sets

  • $x \in S$, $x \notin S$, $\{x \in S \mid P(x)\}$
  • เซตสำคัญ: $\mathbb{R}, \mathbb{Z}, \mathbb{Q}$
  • $A \subseteq B$ ถ้าสมาชิกทุกตัวของ $A$ อยู่ใน $B$
  • $A \times B = \{(a,b) \mid a \in A,\ b \in B\}$

1.3 Relations and Functions

  • Relation $R$ จาก $A$ ไป $B$ คือ subset ของ $A \times B$
  • Function คือ relation ที่ทุก $x \in A$ มีคู่เดียว
  • $F(x)$ คือค่าเดียวที่ $x$ สัมพันธ์กับ $F$
  • Arrow diagram: วาดลูกศรแสดงความสัมพันธ์
